- ස/ශ -> න rule: After 'ස' (sayanna) and 'ශ' (sayanna), the dental 'න' is required (e.g., අත්සණ -> අත්සන, ප්‍රශ්ණ -> ප්‍රශ්න, හසුණ -> හසුන, දේශණ -> දේශන).

- Sibilant confusion: විශේශ -> විශේෂ, දූසනය -> දූෂණය.
